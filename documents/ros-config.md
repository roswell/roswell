
ros-config - get and set options
# Synopsis

**ros [options] config** [SUBCOMMAND ARGS...]

# Description

Without any arguments, it shows all variables and it's value.

show VAR
 
  : set the VALUE to the TARGET.


set VAR VALUE
 
  : set the VALUE to the TARGET.
